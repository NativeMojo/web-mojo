Programmatic Usage

Features

Navigation

  • Click on images or arrow buttons to navigate
  • Keyboard shortcuts: ← → for navigation, Esc to close
  • Image counter shows current position

Display Modes

  • Fit to Screen: Images scale to fit viewport
  • Original Size: Images at natural resolution
  • Click directly on image to toggle between modes
  • Scroll to navigate large images in original size mode

Code Examples

// Import
import LightboxGallery from '../src/lightbox/LightboxGallery.js';

// Single image
LightboxGallery.show('./portal/media/monkey.png');

// Gallery with multiple images
const images = [
  { src: './portal/media/monkey.png', alt: 'Monkey 1' },
  { src: './portal/media/monkey.png', alt: 'Monkey 2' },
  { src: './portal/media/monkey.png', alt: 'Monkey 3' }
];

LightboxGallery.show(images, {
  startIndex: 0,
  showCounter: true,
  fitToScreen: true
});
