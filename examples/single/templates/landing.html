About MOJO

A modern, lightweight JavaScript framework for building professional web applications.

  • Component-Based Architecture: Build reusable, maintainable components
  • Bootstrap 5 Native: Professional UI without custom CSS
  • ES6+ Modern JavaScript: Use the latest language features
  • Minimal Dependencies: Keep your bundle size small
Quick Example
// Create a new app
const app = new WebApp({
    name: 'My App',
    layout: 'portal',
    container: '#app'
});

// Add pages
app.addPage(HomePage);
app.addPage(DashboardPage);

// Start the app
await app.start();
