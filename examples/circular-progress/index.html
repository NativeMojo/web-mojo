Basic Progress

  containerId: 'progress-container'

Arc Styles (Gap)

Create gauge-style indicators with gap angles

const progress = new CircularProgress({
  value: 75,
  gap: 90, // Gap in degrees (creates arc)
  containerId: 'progress-container'
});
await progress.render();
45° Gap
90° Gap (3/4 circle)
180° Gap (half circle)

Rotation Angles

Change where the progress starts

const progress = new CircularProgress({
  value: 75,
  rotation: 0, // Starting angle in degrees
  containerId: 'progress-container'
});
await progress.render();
0° (Right)
90° (Bottom)
180° (Left)
270° (Top)
