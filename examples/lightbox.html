Programmatic Usage

Code Examples

// Single image
LightboxGallery.show('https://picsum.photos/800/600?random=1');

// Single image with alt text
LightboxGallery.show({
  src: 'https://picsum.photos/800/600?random=1',
  alt: 'Beautiful landscape'
});

// Array of image URLs
const images = [
  'https://picsum.photos/800/600?random=2',
  'https://picsum.photos/800/600?random=3',
  'https://picsum.photos/800/600?random=4'
];

LightboxGallery.show(images);

// Array of image objects with alt text
const imagesWithAlt = [
  { src: 'image1.jpg', alt: 'First image' },
  { src: 'image2.jpg', alt: 'Second image' },
  { src: 'image3.jpg', alt: 'Third image' }
];

LightboxGallery.show(imagesWithAlt, {
  startIndex: 1, // Start at second image
  showCounter: true,
  showNavigation: true
});
