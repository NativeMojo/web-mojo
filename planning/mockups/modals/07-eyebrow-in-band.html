// mockup 07 / eyebrow-in-band

Eyebrow-in-Band.

Four ways to put the type label INSIDE the hero band instead of below it. The band gets taller (~28–32px) and carries the eyebrow text directly. Removes the duplication problem entirely — there's only ever one type label, and it sits inside the colored region that already named the type. Compare against Mockup 05 where the band was just a thin slab and the eyebrow lived below.

// variant A — solid band, centered white eyebrow

Most "modal banner" feeling. Strong type signal — you can't miss what kind of message this is. Warning uses black text for AAA contrast on yellow.

// my read

Variant A (solid band + white eyebrow, centered) is the most readable and confident — the eyebrow has weight without being decorative. It also frees the title region completely so headlines like "The operation failed" can stand alone without needing smart-suppression logic. Variant D (newspaper-style with dashes) is the most distinctive but might be too editorial for a system framework. Variant B (tinted light) is fine but reads as "softer banner" — could feel underweight for an error/danger alert. Variant C (left-aligned) creates an asymmetry I don't think pays off.
