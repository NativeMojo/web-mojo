Modal redesign — pick a direction.

Four distinct visual directions for WEB-MOJO's typed alerts, confirms, and form dialogs. Each mockup is a complete “design world” showing the same three surfaces (typed alerts × 4 / confirm / form) so you can compare them on equal footing. The accent color shifts per type in every direction — info / success / warning / error.

const --mojo-dialog-accent = var(--bs-primary); /* customizable per portal */
// 01

Editorial Minimal

Refined, editorial. Thin colored left bar plus an uppercase micro-eyebrow. Fraunces serif title; restrained accent use. Reads "design system," not consumer app.

3px left bar uppercase eyebrow Fraunces + Manrope internal X (square)
// 02

Soft Centered

Closer to your purple "Change your password" reference. Centered title + subtitle. Pill buttons. Soft tinted background and a colored badge at the top. Friendly, consumer-feeling.

centered layout tinted card bg circle type badge pill buttons
// 03

Hero Band

A 6px colored band along the top of the card carries the type. No illustrations. Plus Jakarta Sans + JetBrains Mono micro-labels. Structured, technical, design-system-feeling.

6px top band icon tile + eyebrow JetBrains Mono labels internal X
// 04

Toast Evolved

Take the existing toast styling — left bar + colored circle icon — and grow it up into a modal. Modal and toast read as one family. The most system-coherent and most minimal of the four.

5px left bar colored circle icon toast.css color tokens tiny ghosted X
Each mockup uses Bootstrap 5.3 + Bootstrap Icons via CDN — open any of them standalone in a browser. The chosen direction will be ported into src/core/css/core.css and the existing Modal / Dialog JS surface (no API changes required).
