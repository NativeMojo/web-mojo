ListView · group-divider mockups

Visual treatments for the synthetic header rows that ListView's groupBy primitive inserts between groups of items. Each variant uses identical item markup so the difference you see is only the divider. Toggle theme to compare in light + dark.

A · Subtle primary strip · centered label

your request

Faint --bs-primary-bg-subtle band runs the full width of the items column; small uppercase text centered inside. Reads like an iOS-style date band but tinted with the brand accent. Strong "section break" affordance — a band of color is unmistakable. Trade-off: imposes brand color on every group, which can feel busy in long lists or in apps that don't use the framework's primary as their accent.

B · Typographic mark + fading rule

currently shipped

Sharp 4×4 brand-tinted square + bold uppercase emphasis-color label + directional gradient hairline that fades to nothing on the right. Restrained, typographic, plays well with any consumer's brand. Trade-off: less visually punchy than a band — readers skim more easily but the section break is quieter.

C · Inline chip on muted background

Small pill-shaped label on the framework's --bs-secondary-bg tint, sits inline above each group with no rule or band. Light-touch, conversational, very modern — see Linear's grouped views. Trade-off: the section break relies on white space alone; in dense lists the pill can feel orphaned.

D · Centered label between symmetric rules

Editorial fieldset-legend treatment — label centered, identical hairlines on both sides. Distinctive, designed-feeling. Trade-off: the symmetry can feel formal / magazine-y; works best when groups are infrequent (long sections), feels heavy when groups are short.

E · Full-width neutral band, left-aligned

Same band concept as A but neutral (--bs-tertiary-bg) instead of brand-tinted, label left-aligned instead of centered. Closer to iOS Mail / Things 3 section headers. Trade-off: monochrome — the brand color isn't used to signal section, so the divider is purely tonal. Less character but more universally fitting.

F · Bold heading, no chrome

The most restrained: a slightly larger bold label sitting on its own with breathing room. No fills, no rules, no marks. The whitespace is the divider. See Slack channel groups, Notion list view. Trade-off: cleanest possible look but easy to miss in scannable feeds; no visual chrome to anchor the eye on long lists.
