ListView · group-divider mockups

Visual treatments for the synthetic header rows that ListView's groupBy primitive inserts between groups of items. Each variant uses identical item markup so the difference you see is only the divider. Toggle theme to compare in light + dark.

Recommended directions picked from your review · multiple supported via groupHeaderStyle

E (centered) · Neutral band, label centered

proposed default

Same band concept as E (neutral --bs-tertiary-bg tint, full width) but with the label centered instead of left-aligned. Combines E's universality (no brand-color imposition) with A's centered-anchor energy. Strongest "section break" affordance of the recommended set.

B · Typographic mark + fading rule

your second pick

Sharp 4×4 brand-tinted square + bold uppercase emphasis-color label + directional gradient hairline that fades to nothing on the right. Restrained, typographic, plays well with any consumer's brand. Note: this mockup uses the corrected inset alignment (mark + label aligned with item icons below) — the currently-shipped CSS has zero horizontal padding, so the square hangs off the left edge of the card. Production CSS will be fixed to match this mockup when this variant ships.

E · Full-width neutral band, left-aligned

Same band concept as the centered version above but label left-aligned (closer to iOS Mail / Things 3 section headers). For consumers who prefer the section break to flow with the items' left edge rather than centered. Same neutral --bs-tertiary-bg — no brand-color imposition.

D · Centered label between symmetric rules

Editorial fieldset-legend treatment — label centered, identical hairlines on both sides. Distinctive, designed-feeling. Trade-off: the symmetry can feel formal / magazine-y; works best when groups are infrequent (long sections), feels heavy when groups are short.

rejected

Faint --bs-primary-bg-subtle band, centered uppercase label. Strong section break but imposes brand color on every group; in dark theme the band reads as a saturated navy strip and starts to feel busy in long lists.

C · Inline chip on muted background

rejected

Small pill-shaped label sitting alone on whitespace. Light-touch and modern (Linear-style) but the chip can be confused with item content / status badges, and the section break relies on whitespace alone — too quiet for chronological feeds.
